# Sample Inputs

Use this folder for small local test samples.

Rules:

- Use only images you have permission to use.
- Prefer consented test images from your own team.
- Do not commit private biometric data unless everyone explicitly agrees.
- For public GitHub, use synthetic samples or heavily anonymized evidence images.
